# Sistema de Avaliação ISO 27002:2013

Projeto acadêmico com foco na aplicação prática dos 114 controles da ISO 27002.

## Objetivo
Transformar a norma em uma ferramenta utilizável para avaliação de conformidade organizacional.

## Estrutura
- 14 domínios
- 114 controles
- Sistema de avaliação
- Dashboard de conformidade
